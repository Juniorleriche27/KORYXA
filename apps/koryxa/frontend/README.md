# KORYXA Frontend

Frontend public de KORYXA, la première plateforme d'orchestration IA en Afrique.

## Vitrine publique

Routes principales :

```txt
/               Accueil
/ecosysteme     Écosystème
/produits       Produits autonomes
/cas-usage      Orientation par besoin
/partenaires    Partenaires
/a-propos       À propos
/contact        Contact
```

## Accès central

Le bouton `Compte KORYXA` pointe vers l’accès central public KORYXA via `NEXT_PUBLIC_KORYXA_ACCOUNT_URL`.

## Développement

```bash
npm run dev
npm run build
```
